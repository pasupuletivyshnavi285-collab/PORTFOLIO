# Python ATM Program

This is a simple ATM program written in Python.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Exit from ATM system

## Technologies Used
- Python

## How to Run
1. Download the file `atm_program.py`
2. Run using Python:
   python atm_program.py
